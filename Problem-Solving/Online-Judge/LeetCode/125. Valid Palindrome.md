# Valid Palindrome
[LeetCode](https://leetcode.com/problems/valid-palindrome/)

## Problem Description

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers.

Given a string s, return true if it is a palindrome, or false otherwise.

 
 
## Example & constraints
Example 1:
```
Input: s = "A man, a plan, a canal: Panama"
Output: true
```
Explanation: "amanaplanacanalpanama" is a palindrome.
Example 2:
```
Input: s = "race a car"
Output: false
```
Explanation: "raceacar" is not a palindrome.
Example 3:
```
Input: s = " "
Output: true
```
Explanation: s is an empty string "" after removing non-alphanumeric characters.
Since an empty string reads the same forward and backward, it is a palindrome.
Constraints:

1. 1 <= s.length <= 2 * 105
2. s consists only of printable ASCII characters.

## Solution
```cpp
//3ms solution
class Solution {
public:
    bool isPalindrome(string s) {
        string palindrome="";
        
        for(int i=0;i<s.size();i++){
            if(isalnum(s[i]))
                palindrome.push_back(tolower(s[i]));

        }
        int size = palindrome.size()-1;
        for(int i=0;i<size;i++){
            if(palindrome[i]!=palindrome[size-i]){return false;}
        }
        return true;
    }
};
``` 
```cpp
//1ms solution
class Solution {
public:
    bool isPalindrome(string s) {
        int left = 0 , right = s.size() -1;
        while(left<=right){
            if(!isalnum(s[left])){left++;continue;}
            else if(!isalnum(s[right])){right--;continue;}

            if(tolower(s[left])!=tolower(s[right]))return false;

            left++;right--;

        }
        return true;
    }
};
```
## Explanation
*  **3ms solution**
    *  3ms的版本會用到兩個迴圈，一個是把所有字母都轉換成小寫，一個是檢查是否為回文。
    *  但是真的很慢，因為每次都要檢查是否為字母，而且不太好想

*  **1ms solution**
    *  1ms的版本會用到一個迴圈，一個是檢查是否為字母，一個是檢查是否為回文。
    *  雙指針的方式比較好用，因為回文無非就是從開頭與結尾對比，如果不是字母或數字就跳過，指針往中間移動，只會用到一次迴圈


## Time Complexity
*  Solution 1: time complexity $O(n)$, space complexity $O(n)$
*  Solution 2: time complexity $O(n)$, space complexity $O(1)$
