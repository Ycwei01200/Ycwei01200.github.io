# Power of Two
[LeetCode](https://leetcode.com/problems/power-of-two/)

## Problem Description

Given an integer n, return true if it is a power of two. Otherwise, return false.

An integer n is a power of two, if there exists an integer x such that n == $2^x$.

## Example

Example 1:
```
Input: n = 1
Output: true
```
Explanation: 20 = 1
Example 2:
```
Input: n = 16
Output: true
```
Explanation: 24 = 16
Example 3:
```
Input: n = 3
Output: false
```
## Solution
```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        return (n>0) && (n&(n-1))==0;
    }
};
```
## Explanation

這題是位元運算題目，判斷是不是2的指數，那2的指數的二進位表示法只有一個1，例如1(0001),2(0010),4(0100),8(1000)，那n&(n-1)可以把n的最後一個1拿掉，如果n是2的指數，那n&(n-1)就會是0，所以就能判斷是不是2的指數

|   n   | Binary (n) | Binary (n-1) | n & (n-1) |   Result   |
| :---: | :--------: | :----------: | :-------: | :--------: |
|   1   |    0001    |     0000     |   0000    |  0 (True)  |
|   2   |    0010    |     0001     |   0000    |  0 (True)  |
|   4   |    0100    |     0011     |   0000    |  0 (True)  |
|   8   |    1000    |     0111     |   0000    |  0 (True)  |
|  15   |    1111    |     1110     |   1110    | 14 (False) |

如果要判斷是不是2指數可以拿這個來快速解

## Time Complexity
* $O(1)$
