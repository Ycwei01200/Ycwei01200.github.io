# Longest Palindrome

[LeetCode](https://leetcode.com/problems/longest-palindrome/)

## Problem Description

Given a string s which consists of lowercase or uppercase letters, return the length of the longest palindrome that can be built with those letters.

Letters are case sensitive, for example, "Aa" is not considered a palindrome.
 
## Example & constraints
Example 1:
```
Input: s = "abccccdd"
Output: 7
```
Explanation: One longest palindrome that can be built is "dccaccd", whose length is 7.
Example 2:
```
Input: s = "a"
Output: 1
```
Explanation: The longest palindrome that can be built is "a", whose length is 1.
Constraints:

*   1 <= s.length <= 2000
*   s consists of lowercase and/or uppercase English letters only.
  

## Solution
```cpp
class Solution {
public:
    int longestPalindrome(string s) {
        unordered_map<char,int> count;
        for(auto&c:s)count[c]++;
        int count1=0;
        bool odd = false;
        for(int i=0;i<count.size();i++){
            if(count[i] % 2 !=0){count1+=(count[i]-1);odd = true;}
            else                count1+=count[i];
        }
        if(odd)count1++;
        return count1;
    }
};
```

## Explanation
*  這題其實很簡單，如果字母的數量是偶數，經過重新排列，一定是回文，但如果字母的數量是奇數，最多只能有一個字母在中間，其他都要成對。
*  那在記數上，你可以選擇用兩個`vector`存大小寫，也可以用`map`，然後偶數部分直接加，奇數部分減1再加，最後如果`odd`為true，代表有奇數，就加1，這個會成立的部分都是假設為偶數，如果實際上有奇數，那一定會有一個字母放在中間，滿足最大回文長度。
  
## Time Complexity
*  $O(n)$
## Space Complexity
*  $O(1)$
