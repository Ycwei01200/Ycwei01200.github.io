# Isomorphic Strings
[LeetCode](https://leetcode.com/problems/isomorphic-strings/)

## Problem Description

Given two strings s and t, determine if they are isomorphic.

Two strings s and t are isomorphic if the characters in s can be replaced to get t.

All occurrences of a character must be replaced with another character while preserving the order of characters. No two characters may map to the same character, but a character may map to itself.

## Example & Constraints

Example 1:
```
Input: s = "egg", t = "add"

Output: true
```
Explanation:

The strings s and t can be made identical by:

Mapping 'e' to 'a'.
Mapping 'g' to 'd'.
Example 2:
```
Input: s = "foo", t = "bar"

Output: false
```
Explanation:

The strings s and t can not be made identical as 'o' needs to be mapped to both 'a' and 'r'.

Example 3:
```
Input: s = "paper", t = "title"

Output: true
```
**Constraints**

*   1 <= s.length <= $5 * 10^4$
*   t.length == s.length
*   s and t consist of any valid ascii character.

## Solution
```cpp

class Solution {
public:
    bool isIsomorphic(string s, string t) {
        unordered_map<char,char> alphabet;
        unordered_map<char,char> alphabet2;
        alphabet.clear();
        for(int i=0;i<s.size();i++){
            if(alphabet[s[i]]=='\0'&&alphabet2[t[i]]=='\0'){
                    alphabet[s[i]] = t[i];
                    alphabet2[t[i]] = s[i];
            }
            else if(alphabet[s[i]]==t[i] && alphabet2[t[i]]==s[i]) continue;
            else return false;
        }
        return true;
    }
};
```

## Explanation
### Solution Analysis
* 這題目根本是hash模板題，就是說假設我們有兩個字串s和t，我們要檢查s和t是否是同構的，就是說s和t的字元可以一一對應，並且每個字元的對應關係是唯一的，就是說如果s的字元'a'對應到t的字元'b'，那麼s的其他字元就不能再對應到t的其他字元了。
*   **Example 1:** `s = "egg", t = "add"`
    *   `'e' -> 'a'`
    *   `'g' -> 'd'`
    *   每個字元都有對應到唯一的字元，所以是同構的。
*   **Example 2:** `s = "foo", t = "bar"`
    *   `'f' -> 'b'`
    *   `'o' -> 'a'`
    *   `'o' -> 'r'`
    *   `'o'` 同時對應到多個字元 (`'a'` 和 `'r'`)，所以不是同構的。

* 回到程式碼，透過兩個`unordered_map`來存s和t的字元，一個是檢查s對應t的字元，是否唯一，另一個是檢查t對應s的字元，是否唯一，如果都唯一，就是同構的


## Complexity
*   **Time Complexity:** $O(n)$
*   **Space Complexity:** $O(1)$
