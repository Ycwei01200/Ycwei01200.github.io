# Palindrome Number

[LeetCode](https://leetcode.com/problems/palindrome-number/)

## Problem Description

Given an integer x, return true if x is a palindrome, and false otherwise.

* Note : An integer is a palindrome when it reads the same backward as forward.

## Example

```
Input: x = 121
Output: true
```

```
Input: x = -121
Output: false
```

```
Input: x = 10
Output: false
```

## Solution

```cpp
class Solution {
public:
    bool isPalindrome(int x) {
        if(x < 0)return false;
        else{
            string y;y.clear();
            while(x!=0){
                y+=((x%10)+'0');x/=10;
            }
            for(int i = 0 ; i < y.size() ; i++){
                if(y[i]!=y[y.size()-i-1])return false;
            }
            return true;
        }
    }
};


```
## Explanation
*  the first thing is to check whether the number is negative or not
*  if it is negative, return false
*  if it is not negative, convert the number to a string and check whether it is a palindrome or not
*  the reverse of the string is a good way to check whether it is a palindrome or not.
