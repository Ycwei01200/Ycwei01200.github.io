# Valid Anagram
[LeetCode](https://leetcode.com/problems/valid-anagram/)

## Problem Description

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

## Example

Example 1:
```
Input: s = "anagram", t = "nagaram"

Output: true
```
Example 2:
```
Input: s = "rat", t = "car"

Output: false
```
 

Constraints:

*   1 <= s.length, t.length <= $5 * 10^4$
*   s and t consist of lowercase English letters.
## Solution
```cpp
//0ms version
class Solution {
public:
    bool isAnagram(string s, string t) {
        if(s.size()!=t.size())return false;
        int count[27]={0};
        for(int i=0;i<s.size();i++){
            count[s[i]-'a']++;
            count[t[i]-'a']--;
        }
        for(int x=0;x<25;x++){
            if(count[x]!=0)return false;
        }
        return true;
    }
};
```
```cpp
//4ms version
class Solution {
public:
    bool isAnagram(string s, string t) {
        if (s.size() != t.size()) return false;
        sort(s.begin(),s.end());sort(t.begin(),t.end());
        return s==t;
    }
};
```
## Explanation

這題，題目是在說，判斷兩個字串是不是同構的(同字母數量，不同排列組合)，那這就有兩個辦法，你可以用紀錄的，就像我第一個辦法，再來就是用字典排序法，比較不用動腦，但也比較慢


## Time Complexity

### 0ms version
* $O(n)$

### 4ms version
* $O(nlogn)$
