# 461. Hamming Distance

[LeetCode](https://leetcode.com/problems/hamming-distance/description/?envType=problem-list-v2&envId=wpnf8mnm)

## Problem Description

The Hamming distance between two integers is the number of positions at which the corresponding bits are different.

Given two integers x and y, return the Hamming distance between them.

 
## Example & constraints
Example 1:
```
Input: x = 1, y = 4
Output: 2
```
Explanation:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑
The above arrows point to positions where the corresponding bits are different.
Example 2:
```
Input: x = 3, y = 1
Output: 1
```
Constraints:

1. 0 <= x, y <= $2^{31} - 1$
## Solution
```cpp
class Solution {
public:
    int hammingDistance(int x, int y) {
        int s = x^y;
        int count = 0;
        while(s!=0){
            count += (s&1);
            s>>=1;
        }
        return count;
    }
};
```


## Explanation
*  這題，是問說，當指標走過`x`跟`y`時，有幾個位元是不同的，那有兩種方法，第一個就是說，你可以將`decimal`轉成`binary`，然後比對每一位元，第二個就是說，你可以用`XOR`來比對每一位元，因為`XOR`的特性就是說，如果兩個數字都是`1`，結果就是`1`，代表這一位元是相同的，如果`XOR`的結果是`0`，代表這一位元是不同的，剛好可以用來作為判斷的依據。
  
## Time Complexity
*   $O(n)$
## Space Complexity
*   $O(1)$
