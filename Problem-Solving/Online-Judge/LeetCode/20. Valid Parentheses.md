# Valid Parentheses
[LeetCode](https://leetcode.com/problems/valid-parentheses/)

## Problem Description

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

Open brackets must be closed by the same type of brackets.
Open brackets must be closed in the correct order.
Every close bracket has a corresponding open bracket of the same type.

## Example & constraints

Example 1:
```
Input: s = "()"

Output: true
```
Example 2:
```
Input: s = "()[]{}"

Output: true
```
Example 3:
```
Input: s = "(]"

Output: false
```
Example 4:
```
Input: s = "([])"

Output: true
```
Example 5:
```
Input: s = "([)]"

Output: false
```
constraints:

1. 1 <= s.length <= $10^4$
2. s consists of parentheses only '()[]{}'.

## Solution
```cpp
class Solution {
public:
    bool isValid(string s) {
        vector<char> left_stack1;
        for(int i=0;i<s.size();i++){
            if(s[i]=='('||s[i]=='{'||s[i]=='[')left_stack1.push_back(s[i]);
            else if(s[i]==')'||s[i]=='}'||s[i]==']'){
                if(left_stack1.empty())return false;
                else
                {
                    if(s[i]==')'&& left_stack1[left_stack1.size()-1]=='('){left_stack1.pop_back();}
                    else if(s[i]==']'&& left_stack1[left_stack1.size()-1]=='['){left_stack1.pop_back();}
                    else if(s[i]=='}'&& left_stack1[left_stack1.size()-1]=='{'){left_stack1.pop_back();}
                    else return false;
                    
                }
            }
        }
        if(!left_stack1.empty())return false;
        
        return true;
        
    }
};
``` 
## Explanation
*  it is a stack problem, we use array to implement the stack.
*  the stack which follows the FILO principle, so we use the last element to check whether it is the same as the current element.
*  first, we check whether the stack is empty, if it is empty, we return false.
*  if it is the pair, we pop the last element, else we return false. 
## Time Complexity
*  $O(N)$
