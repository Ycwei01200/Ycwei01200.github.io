# Add Binary
[LeetCode](https://leetcode.com/problems/add-binary/)

## Problem Description

Given two binary strings a and b, return their sum as a binary string.

## Example & constraints
Example 1:
```
Input: a = "11", b = "1"
Output: "100"
```
Example 2:
```
Input: a = "1010", b = "1011"
Output: "10101"
```
Constraints:

1. 1 <= a.length, b.length <= 104
2. a and b consist only of '0' or '1' characters.
3. Each string does not contain leading zeros except for the zero itself.

## Solution
```cpp
class Solution {
public:
    string addBinary(string a, string b) {
        reverse(a.begin(),a.end());
        reverse(b.begin(),b.end());
        string c="";
        
        while(a.size()+1!=b.size()+1){
            if(a.size()+1<b.size()+1)a.push_back('0');
            else if(a.size()+1>b.size()+1)b.push_back('0');
        }
        int n=a.size(),m=b.size();
        int carry = 0;
        for(int i=0;i<n;i++){
            int sum = carry + (a[i]-'0')+(b[i]-'0');
            c.push_back((sum % 2)+'0');
            carry = sum/2;
            
        }
        if(carry)c.push_back('1');
        reverse(c.begin(),c.end());
        return c;
    }
};
``` 
## Explanation
*  the first step is to reverse the string `a` and `b`, then we can check whether the length of `a` and `b` is the same, if not, we need to add '0' to the shorter string.
*  then we can use the same method as `Plus One` to solve this problem.
*  second,we should calculate the `carry` and sum of the current digit, then add it to the string `c`.
*  third,we should reverse the string `c` and return it.
## Time Complexity
*  $O(max(n, m))$
