# YKL01 (UVA10055) Hashmat the brave warrior

[UVA](https://onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&page=show_problem&problem=996)

## Solution

```cpp
#include<iostream>
using namespace std;
typedef long long ll;
int main(){
  ll x,y;
  while(cin>>x>>y){cout<<abs(x-y)<<endl;}
  return 0;
}

```

## Explanation

這題，就叫你直接算兩數之差，記得加上絕對值。
